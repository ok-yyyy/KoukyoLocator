# KoukyoLocator

スマートフォンの位置情報と方位センサーを使って、皇居の方向と距離を表示するシンプルな Web アプリです。

[Link2011-Act2/BeastLocator](https://github.com/Link2011-Act2/BeastLocator) にインスピレーションを受けて作成されました。

## Demo

https://ok-yyyy.github.io/KoukyoLocator/

## Features

- 現在地から皇居までの距離を表示
- 端末の向きに応じて、皇居の方向を画像で表示
- iPhone / Android の方位センサーに対応
- Vite + React + TypeScript で構成

## Notes

- 方位センサーに対応したスマートフォンでの利用を想定しています
- PC ブラウザでは正しく動作しない場合があります
- 位置情報や方位情報が取得できない場合は、ブラウザや OS の権限設定を確認してください
